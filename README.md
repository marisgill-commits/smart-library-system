# 🔀 Git Workflow Implemented

## 1️⃣ Branching Strategy
- Created feature-based branches:
  - feature-home
  - feature-books
  - feature-about
  - feature-contact

---

## 2️⃣ Commit History
- Structured and meaningful commits:
  - Add homepage UI
  - Add books page layout
  - Add about section design
  - Add contact form UI

---

## 3️⃣ Pull Request Workflow
- Each feature developed in separate branch  
- Code reviewed via GitHub Pull Requests  
- Merged into main branch after verification  

---

## 4️⃣ Repository Management
- Clean repository structure  
- Organized files and folders  
- Proper version control practices  

---

## ☁️ GitHub Integration
- Repository hosted on GitHub  
- Remote repository linked using Git  
- Regular commits and pushes maintained
